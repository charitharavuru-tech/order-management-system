# Order Management System

This project is a backend REST API built using Java and Spring Boot for managing customer orders in an e-commerce system.

## Technologies Used
- Java
- Spring Boot
- REST APIs
- MySQL
- Maven

## Features
- Create new orders
- Retrieve order details
- Update order status
- Delete orders
- RESTful API design following layered architecture

## Architecture
The application follows a standard layered architecture:
Controller → Service → Repository → Database

## Purpose
This project demonstrates backend development skills including REST API design, object-oriented programming, and database integration using Spring Boot.

## Future Improvements
- Add authentication using Spring Security
- Containerize using Docker
- Deploy on AWS
